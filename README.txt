How to run?
1: start mongodb
2: npm run dev

How to operate?
1: Signup as a user (buyer)
2: goto "test" db of mongodb and change the type of user to admin (sellter)
3: Upload products and again login as a user (buyer) to place orders
4: Stripe credential need to be defined yet so it will stop at checkout
5: Signup as a user and change its type to root from mngodb to view all users and transactions

How many user types?
1: USER as a buyer
2: ADMIN as a seller
3: ROOT as an admin
